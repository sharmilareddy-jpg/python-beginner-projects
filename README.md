# Number Guessing Game 🎮

A clean, text-based terminal game built with Python. The program generates a random number between 1 and 100, and guides the player to the correct answer using high/low hints.

## 🚀 Key Programming Concepts Used
- **Functions:** Isolated code execution for cleaner project architecture.
- **Loops & Conditionals:** Continuous checking via `while` loops and `if/elif/else` statements.
- **Error Handling:** Integrated `try-except` blocks to prevent crashes from invalid user entries.
- **Random Module:** Utilizing `random.randint` for absolute unpredictability.

## 🛠️ How to Play
1. The computer chooses a hidden number between 1 and 100.
2. Enter your guess when prompted in the console.
3. Read the feedback hints ("Too high!" or "Too low!").
4. Keep guessing until you crack the code—your final attempt score will be displayed at the end!

## 📦 How to Run Locally
1. Clone this repository or download the `.py` script file.
2. Open your local terminal or command prompt.
3. Run the script using the following command:
   ```bash
   number guessing.py
   ```
